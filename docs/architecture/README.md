# Architecture Docs

Architecture docs are intentionally minimal in v0.

This starter export includes only public-safe, generic documentation. It does not include private deployment inventory, private provider test history, account-specific usage notes, or live provider incident records.

More architecture docs can be added later for:

- provider boundaries
- model switching
- prompt / context / memory rooms
- usage logging
- debug safety
- deployment data persistence

Use fake data in examples and keep provider secrets server-side.
