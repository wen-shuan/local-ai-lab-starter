Provider Error Guide

How to read the sanitized Last Chat Debug summary. This guide does not change retry or provider behavior.

429 / RESOURCE_EXHAUSTED
Usually quota or rate-limit related. Pause instead of repeatedly retrying.
503 / unavailable or overloaded
The provider may be temporarily unavailable, overloaded, or wrapping a lower-level error as 503.
externalApiCalled
Shows whether this request actually attempted an external provider call. If false, it is usually mock, disabled, or dry-run mode.
Retry
Use only for temporary errors. The app does not auto-retry, which avoids extra provider calls.
Sanitized debug
Last Chat Debug should show only sanitized error details. It should not show API keys, tokens, or secrets.
Invalid selected model / 400
The selectedModelId is not in the server allowlist. The server rejects it and does not call an external provider.
